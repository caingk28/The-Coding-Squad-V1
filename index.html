The Coding Squad

Empower your child with the skills to succeed in the digital age. Our fun and interactive curriculum teaches children to code using Scratch and Python, starting with simple games and progressing to more advanced projects.

Some of the First Games We'll Make

Computational Focus - Loops
Scratch - Maze Game

Children will begin their first game by designing a maze and then creating their own sprite to maneuver around the maze. with a focus on loops children will code their sprite to get around the maze.

Computational Focus - Data types
Scratch - Matching Pairs

To create the matching pairs game, children will design a set of cards using a set of preloaded of images or their own creation. The aim will be to match the pairs, they will use data tpyes to check if the two cards match.

Computational Focus - Conditional Statements
Scratch - Space Invaders

Children will recerate the game Space Invaders. They will design their own spaceship and aliens and will use conditional statements to check for collisions and decide when the game ends

Computational Focus - Variables
Scratch - Snake

Children will design and create a snake game where their sprite has to eat food and grow longer whilst avoding certain blocks, variables are used to store the position of the snake and the location of the food.
